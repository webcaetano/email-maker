### Instalation

```
git clone https://github.com/webcaetano/email-maker.git
cd email-maker
npm i
```

### Commands 

Start development mode
```
gulp serve
```


Send a email test for your inbox.
```
gulp test
```


Create html file of your template 
```
gulp build
// or
gulp 
```
